# 🚀 AI-Powered ATS Resume Analyzer & Career Assistant

An AI-powered full-stack web application designed to help job seekers optimize their resumes, improve ATS compatibility, and generate professional job application documents.

## 🌐 Live Demo

🔗 Live Application: https://ai-powered-ats-resume-analyzer-career-jhwn.onrender.com/

## 📌 Overview

Most resumes are rejected before they ever reach a recruiter due to Applicant Tracking Systems (ATS).

This project helps users:

* Analyze resumes using AI
* Evaluate ATS compatibility
* Match resumes with job descriptions
* Generate personalized improvement suggestions
* Create professional cover letters
* Generate job proposals

## ✨ Features

### 📄 Resume Analysis

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/kanchan874/AI-Powered-ATS-Resume-Analyzer-Career-Assistant.git
```

Move into project directory:

```bash
cd AI-Powered-ATS-Resume-Analyzer-Career-Assistant
```

Install dependencies:

```bash
npm install
```

Create a .env file:

```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_api_key
```

Run the application:

```bash
npm start
```
